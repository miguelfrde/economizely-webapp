## Setup

Make sure you have NVM installed.

Clone this repo and then:

```
$ nvm install 6.1.0
$ cd /path/to/economizely-webapp
$ nvm use
$ npm install
$ npm start
```

## Testing

Right now, the repository doesn't include any test. We are only linting:

```
$ npm run lint
```
